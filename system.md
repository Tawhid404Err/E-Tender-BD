# EGP Automation System - User Guide

## What This System Does

This is a comprehensive **e-Government Procurement (EGP) Automation System** designed to streamline and automate the tender management process for businesses participating in government procurement in Bangladesh. The system helps users manage company registrations, track tenders, create job orders, handle payments, and prepare tender documents.

---

## Core User Features

### 1. **User Account Management**

- **Registration & Login**: Create user accounts with secure authentication
- **Password Recovery**: Reset forgotten passwords via email
- **User Profile**: View and update personal information
- **Wallet System**: Track available balance for purchasing services
- **Role-Based Access**: Different permissions for regular users and administrators

### 2. **Company Registration & Migration**

- **Register Companies**: Add company details for EGP portal participation
- **Company Migration**: Migrate existing company credentials to the system
- **Track Migration Status**: Monitor whether companies are active, waiting, or processed
- **Multiple Company Management**: Handle multiple company registrations under one user account
- **View Company Details**: Access complete company information including contact details, trade licenses, and registration numbers

### 3. **Tender Discovery & Management**

- **Browse Live Tenders**: View all currently active government tenders
- **Search & Filter Tenders**: Find tenders by:
  - Department/Organization
  - Tender category
  - Location/District
  - Tender method (OTM, LTM, OSTETM)
  - Publication date
  - Opening date
- **Tender Details**: View comprehensive tender information including:
  - Tender description and scope of work
  - Budget and pricing
  - Document submission deadlines
  - Opening dates and times
  - Required qualifications
  - Terms of reference (TDS)
- **Tender Statistics**: See counts of available tenders by type
- **Emergency Tender Alerts**: Get notified about urgent tender opportunities

### 4. **Job Order System (Service Requests)**

- **Create Job Orders**: Request services for tender participation including:
  - Tender document preparation
  - Form filling assistance
  - Category assignment
  - Bulk order processing
- **Shopping Cart**: Add multiple services to cart before checkout
- **Order Tracking**: Monitor status of placed orders:
  - Waiting for processing
  - In progress/Working
  - Fulfilled/Completed
  - Cancelled
- **My Orders**: View personal order history and current orders
- **Order Updates**: Receive status updates on job progress

### 5. **Payment Management**

- **Wallet Recharge**: Add funds to user wallet via payment methods (BKash, etc.)
- **Payment History**: View transaction records
- **Promo Codes**: Apply discount codes for special offers
- **Balance Checking**: Monitor available wallet balance before ordering
- **Payment Requests**: Submit payment proof for admin verification

### 6. **Bank Services & Financial Documents**

- **Account Holder Management**: Store and manage bank account details for tender submissions
- **Line of Credit Generation**: Create credit line documents required for tender participation
- **Bank Details**: Maintain banking information for official documentation
- **PDF Generation**: Automatically generate formatted bank documents for tender submissions

### 7. **Tender Preparation Tools**

- **Contact Information Forms**: Prepare company contact details for tender submissions
- **Company Profile Builder**: Create formatted company profiles for proposals
- **Bill of Quantities (BOQ)**: Generate pricing and quantity schedules
- **Tender Dashboard**: Centralized workspace for preparing tender submissions
- **Document Templates**: Use pre-formatted templates for common tender requirements

### 8. **Data Entry & Management**

- **Tender Data Entry**: Input new tender information into the system
- **Update Tender Details**: Modify existing tender information
- **Category Management**: Organize tenders by categories and subcategories
- **Department Management**: Maintain list of government departments and organizations
- **Tender ID Tracking**: Monitor tender identification numbers across different methods

### 9. **Reporting & Analytics**

- **At-a-Glance Dashboard**: View key metrics including:
  - Total active tenders
  - Pending job orders
  - Completed orders
  - Wallet balance
  - Recent order activity
- **Recent Orders**: Quick access to latest service requests
- **Tender Statistics**: Visual breakdown of tenders by category and method
- **By-Pass Reports**: Generate special compliance reports for specific tenders

### 10. **Tutorial & Help System**

- **Tutorial Categories**: Organized learning resources
- **Step-by-Step Guides**: Instructions for using various system features
- **Category-Based Learning**: Find help based on specific topics

### 11. **Administrative Functions** (Admin Users Only)

- **User Management**: View and manage all registered users
- **Company Verification**: Approve and verify company migration requests
- **Service Creation**: Define available services and their pricing
- **Promo Code Management**: Create and manage discount codes
- **Payment Processing**: Review and approve payment requests
- **Data Quality Control**: Review and correct tender data entries
- **System Configuration**: Manage departments, categories, and system settings

### 12. **Document Management**

- **PDF Generation**: Automatically create printable documents for:
  - Tender proposals
  - Bank guarantees
  - Credit line certificates
  - Company profiles
- **File Upload**: Attach supporting documents to orders and registrations
- **Download Center**: Access completed documents and reports

## User Workflows

### For Regular Users (Businesses):

1. **Register** an account
2. **Add company** details and complete migration
3. **Browse tenders** matching your business
4. **Create job orders** for services you need
5. **Make payment** via wallet
6. **Track order status** until completion
7. **Download** completed documents

### For Administrators:

1. **Monitor** all tender entries and user activities
2. **Verify** company registrations
3. **Process** job orders
4. **Manage** payments and promo codes
5. **Maintain** system data quality
6. **Support** users with their requests

---

## System Scope

This system **helps users**:

- Find and track government procurement opportunities
- Manage company credentials for EGP portal
- Request document preparation services
- Generate required financial documents
- Organize tender submission materials
- Track service requests and payments

This system **does not**:

- Automatically submit tenders on behalf of users
- Guarantee tender wins
- Replace the official government EGP portal
- Make decisions about which tenders to bid on
- Provide legal or business consulting services
